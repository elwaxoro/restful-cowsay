Aww yiss! Cowsay!

All requests should be sent as REST parameters.

Examples

  • http://localhost:4567/say?msg=Hi%20Mom!
  • http://localhost:4567/say?msg=Thinking%20Thinking&cowthink
  • http://localhost:4567/say?msg=Penguins!&f=tux

See this for general instructions: cowthink man page
